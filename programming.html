Tactical Path Planning in Starcraft II

Technologies: C++, Starcraft II API

Using the StarCraft II API, I assisted in the implementation of tactical path planning using Influence Maps to find the safest and shortest path for allied units within the game. By developing Influence Maps, the allied units are able to utilize the data generated from the influence maps to make meaningful responses to the environment. Influence is altered based on certain characteristics the enemy usits possess, for instance, weapon range, attack type, and unit health. In short, the stronger the enemy units are, the more influence the possess over the area they reside in. Based on this, allied units will either choose to attack or avoid the enemy, if their own influence is not equal to or greater than the enemy.

Foodini

Technologies: Swift, XCode

Within XCode, I created the Detailed Item view for this application. Based on the information received from a food database API, I was able to flag certain allergens within this view. Additionally, based on the items expiration date and current date, I was able to track which pantry items were expired or expiring soon.
